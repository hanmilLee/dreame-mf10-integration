# sessions/

Log per sessione di lavoro. Una nota per sessione, anche breve.

## Convenzioni
- Naming: `YYYY-MM-DD-<slug>.md` (es. `2026-05-22-bootstrap.md`). Se più sessioni nello stesso giorno: `-a`, `-b`.
- Template minimo:
  ```
  # <data> — <topic>
  ## Contesto
  ## Cosa fatto
  ## Decisioni
  ## Blocchi / domande aperte
  ## Prossimi passi
  ```
- Riferire piani in `plans/` e snapshot in `research/` con link relativi.
- Non duplicare la spec: linka, non copiare.
